Flexbox

Qu'est-ce que c'est?

Le principe de la mise en page avec Flexbox est simple : vous définissez un conteneur, et à l'intérieur vous placez plusieurs éléments. Imaginez un carton dans lequel vous rangez plusieurs objets : c'est le principe !

Sur une même page web, vous pouvez sans problème avoir plusieurs conteneurs. Ce sera à vous d'en créer autant que nécessaire pour obtenir la mise en page que vous voulez.

Pour commencer, il suffit d'ajouter à la propriété CSS de votre conteneur:
display:flex;


Les deux axes de Flexbox

Lorsqu'on travaille avec flexbox on doit penser selon deux axes, ça nous permet d'agencer ces éléments selon l'axe horizontal ou vertical. l'axe principal est défini par la propriété flex-direction l'autre axe est donc perpendiculaire. Tout ce qu'on fait avec flexbox. Les axes sont très importants.
flex-direction:row;
flex-direction:column;
flex-direction:row-reverse;
flex-direction:column-reverse;
